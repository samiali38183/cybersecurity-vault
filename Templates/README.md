# 📝 Templates

Documentation templates used across this repository for consistency.

| Template | Used for |
|----------|----------|
| `CTF-Writeup-Template.md` | HackTheBox / TryHackMe / CyberDefenders write-ups in `07-CTF-Writeups/` |
| `Phishing-Analysis-Template.md` | End-to-end phishing sample analyses with IOC extraction and ATT&CK mapping |
| `Vulnerability-Report-Template.md` | Vulnerability scan write-ups with CVSS, EPSS, KEV-aware prioritization |

Templates use YAML frontmatter so notes generated from them are queryable and ATT&CK-tagged consistently.
